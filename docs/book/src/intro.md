# Introduction

This is the reference manual for the Rust implementation of Dynomite. It
documents the embeddable engine crate and the `dynomited` server binary.

The manual is organised top-down: architecture, configuration, the
protocols Dynomite speaks and frames between nodes, the transports it
runs over, security, operations, and the embedding API for in-process
use.

For the staged plan and the operating manual that drives the work, see
`PLAN.md` and `AGENTS.md` at the repository root.
